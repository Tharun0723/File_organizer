# File Organizer 📁

A Python automation script that organizes files in a folder based on their file types.

## Features
- Automatically sorts files into folders
- Supports images, documents, videos, music, and archives
- Creates folders automatically if they don't exist

## Technologies Used
- Python
- os module
- shutil module

## How to Run

1. Place files inside a test folder
2. Update folder_path in file_organizer.py
3. Run the program:

python file_organizer.py

## Example

Before:
image.jpg
video.mp4
document.pdf

After:
Images/image.jpg
Videos/video.mp4
Documents/document.pdf
